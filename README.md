# RehabGuiden (statisk webbplats)

Denna leverans är byggd för statisk hosting via GitHub och körning via Microsoft Azure Static Web Apps.

## Struktur
- `index.html` – startsida
- `style.css` – extern CSS (ligger i samma katalog som index.html)
- `js/main.js` – enkel JS (aktiv meny + mjuk scroll)
- `artiklar/` – längre bloggliknande artiklar
- `program/` – rehabprogram per område
- `liniment-tester/` – katalog skapad men lämnas tom enligt kravspec
